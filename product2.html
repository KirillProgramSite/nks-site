Боксы радиационно-защитные

Назначение

Боксы перчаточные радиационно-защитные предназначены для работ с открытыми радиоактивными веществами (расфасовка, дозирование, проведение химических реакций, приготовление меченых соединений и т.д.).

Конструкция

  • Герметичный корпус из коррозионностойкой стали с закругленными углами стенок;
  • Наклонное смотровое окно из органического стекла или поликарбоната;
  • Перчаточные проемы с камерными перчатками;
  • Форкамера в боковой стенке для передачи предметов;
  • Система приточно-вытяжной вентиляции с регулировкой воздухообмена;
  • Тягонапоромер для контроля разрежения;
  • Подвод коммуникаций:
    • - горячая и холодная вода
    • - сжатый воздух
    • - бытовой газ
    • - подача реагентов
    • - система вывода твердых и жидких радиоактивных отходов
  • Электрификация:
    • - встроенный светильник
    • - герметичные розетки
  • Управление коммуникациями с помощью маховичков на фасадной стороне.
